In database management system, there are total three levels. Physical level means database, logical level means actual table in tabular format and view level means different views as per their priority. Data Independence means users and datashould not directly interact with each other. One is the physical level, the other one is the logical level and finally we have the view level. The applications actually do not depend on the physicallevel or the physical Schema. So obviously the interaction with the system is done at this level. We know all data are stored in a central place. The normal user will be able to access his own data. A teller may access some more data than the normal user. The application does not need to be on this level to work.
