Levels of Data Abstraction are associated with Conceptual Level (Logical Level) and And View Level (External Schema) Logical Level supports Physical Data Independence. View Level enables User Interaction. For instance, hiding Complexity provides Essential Details. Physical Level (InternalSchema) is Completely handled by Database Administrator (DBA) Furthermore, physical Data Independence provides Independence from Physical Level. Data AbStraction supports Primary Goal of Database. Levels of Abstractions includes Physical Level and Hiding Complexity. View level contains Multiple Views. For example, hiding complexity provides essential details about a database. For more information on how to use these levels, please visit: http://www.mysql.com/data-abstraction/levels-of-data-straction-and-hiding-complexity-for-mysql-schema.html.
