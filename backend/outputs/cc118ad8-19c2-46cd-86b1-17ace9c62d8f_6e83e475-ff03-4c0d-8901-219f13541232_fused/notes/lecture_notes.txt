2. Data Independence
Definition: Users and data should not directly interact with each other.

3. Levels Of Data Abstraction
Definition: Total three levels in database management system: View level, conceptual level, and physical level.
  - Physical Level (Internal Schema): The lowest level of data abstraction, defining how data are stored and its location in memory, completely handled by the database administrator.
  - Conceptual Level (Logical Level): The level that defines what data are stored in the database, relationships among data, and the structure of the entire database, handled by DBA, programmer, or developer.
  - View Level (External Schema): The level that tells the application how data should be shown to the user, providing multiple views of the same database for user interaction.
  - Data Independence: Users and data should not directly interact with each other.
