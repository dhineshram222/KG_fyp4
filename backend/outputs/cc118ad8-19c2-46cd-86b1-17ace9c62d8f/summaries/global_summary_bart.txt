In database management system, there are total three levels. That is physical level and physical level is directly connected to the database. There is multiple views of the same database. Data independence means users and data should not directly interact with each other. The next level of database data abstraction is conceptual level or logical level. logical level defines that what data are stored into the database and what are the relationships exist among this data. Logical level decide the structure of entire database. View level generally tells the application about how data should be shown to the user. Physical level means database, logical level means actual table in tabular format and view level means different views as per their priority. So this is, these are the examples of data abstraction in database management system. Data Abstraction refers to the process of hiding irrelevant details from the user.
