1. Network Topologies
Definition: Refers to the arrangement of different elements like nodes, links, and devices in a network; defines how these components are connected and interact with each other.
1.1 Types of Network Topologies
Categories of network topologies, including point-to-point, bus, ring, star, mesh, tree, and hybrid.
  - Point to Point Topology: A network topology that connects two nodes or devices, where one acts as a sender and the other as a receiver, providing high bandwidth and high speed.
  - Bus Topology: A topology where every device is connected to a single coaxial cable called the backbone cable, allowing only one device to send information at a time.
  - Ring Topology: A topology where all PCs are connected with each other in the form of a ring, and data moves from one node to another either clockwise or anticlockwise.
  - Star Topology: A configuration where all nodes are connected to a central hub using a communication link, with the hub functioning as a server.
  - Mesh Topology: A topology where all computers are interconnected with each other through various redundant connections, providing multiple paths for sending and receiving data.
  - Tree Topology: A topology that combines characteristics of bus and star topologies, connecting computers in a hierarchical fashion with point-to-point or point-to-multipoint links.
  - Hybrid Topology: A combination of various different topologies, created based on the specific requirements of the network.
  - Network Topology Applications: Used in network designing algorithms, routing algorithms in operating systems, image recognition, data compression, and database management systems.

7. Csma Technique
Definition: A technique used in Bus Topology to handle data flow and guarantee data integrity.
  - Collision Avoidance (CSMA/CA): A CSMA technique where the sender waits until the transmission media becomes idle if it is busy.
  - Collision Detection (CSMA/CD): A CSMA technique where the sender stops transmitting data if a collision is detected, working on 'recovery after the collision'.

8. Ethernet Lan Cable
Definition: An example of Bus Topology commonly used in LAN and small offices.

9. Token Passing
Definition: A method of transferring data in Ring Topology where the sender modifies a token with the destination address and data, which is passed until it matches the destination.

10. Hub (Star Topology)
