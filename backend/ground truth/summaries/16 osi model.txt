The OSI (Open System Interconnection) model, developed by ISO, is a seven-layer framework designed to ensure accurate and smooth communication across networks. It is divided into hardware layers (Physical, Data Link, Network), software layers (Session, Presentation, Application), and the Transport layer, which serves as the "heart" of the model by connecting the two groups.
The Application layer enables direct user interaction through protocols like HTTP and FTP, while the Presentation layer handles data translation, encryption, and compression. The Session layer manages connection establishment and authentication, leading to the Transport layer, which ensures reliable process-to-process delivery using segmentation and protocols like TCP or UDP. At the lower levels, the Network layer manages routing and logical (IP) addressing, while the Data Link layer handles framing and physical (MAC) addressing. Finally, the Physical layer is responsible for transmitting raw bits as signals over various wired or wireless media.
